# Matrix Inversion
The inverse of a Matrix is strictly only defined for [square matrices]().

$$\large\tag{1}
\mathbf{A}^{-1} \mathbf{A}=\mathbf{I}=\mathbf{A} \mathbf{A}^{-1}
$$
