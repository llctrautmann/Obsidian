# Functions
## Defining a function
I want to start with a simple definition of a function: 

$$
\begin{align}
f: \mathbb{R}^D &\to \mathbb{R} \\
\boldsymbol{x} &\mapsto f(\boldsymbol{x})
\end{align}
$$

A functions is a mapping from the Domain to the Codomain.
