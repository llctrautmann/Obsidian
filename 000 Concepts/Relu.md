# Relu

ReLU returns 0 for any negative input, and returns the input value itself for any non-negative input.

$$\large\tag{1}

\text{ReLU}(x) = \max(0, x) =

\begin{cases}

0 & \text{for } x < 0 \\

x & \text{for } x \geq 0

\end{cases}

$$
