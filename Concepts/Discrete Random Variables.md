# Discrete Random Variables

Discrete Random Variables are mappings from events in the sample space $\Omega$ onto the real number line $\mathbb{R}$. 

> [!Definition]-
> A random variable $X$ is a function $X: \Omega \mapsto \mathbb{R}$
